Manage product launches, campaign performance, inventory signals, conversion metrics, and checkout opportunities.

## Features

- **Product Management** — Add, edit, and track product listings
- **Campaign Tracking** — Monitor marketing campaigns and conversion rates
- **Inventory Signals** — Stock levels, reorder alerts, and availability
- **Conversion Metrics** — Track funnel performance and checkout rates
- **Search & Filter** — Find products by status, category, or performance
- **Analytics View** — Sales distribution and campaign breakdown
- **Import / Export** — JSON-based product data portability

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Markup | HTML5 |
| Styling | CSS3 (Custom Properties, Grid, Flexbox) |
| Logic | Vanilla JavaScript (ES2022) |
| Storage | LocalStorage |
| Server | Python HTTP server (dev) |

## Getting Started

### Prerequisites

- Python 3.x (for dev server)
- Modern web browser

### Installation

```bash
git clone https://github.com/vizvasanlya/First.git
cd First/vizvasanlya-ecommerce-storefront
```

### Run

```bash
npm start
```

Open [http://localhost:5173](http://localhost:5173)

### Docker

```bash
docker compose up --build
```
